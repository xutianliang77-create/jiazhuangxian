# ASK-071 参考答案

## 三种切法

### 1. CLI 内 slash（最快）

```
> /provider use lmstudio
provider: switched to lmstudio (qwen/qwen3.6-35b-a3b)
```
切回：`/provider use openai`

### 2. 改配置文件 + 重启

```bash
# 编辑 current 字段
$EDITOR ~/.codeclaw/providers.json
#   "current": "lmstudio"

# 重启 codeclaw
```

### 3. Web UI（如已起 web）

PATCH `/v1/web/providers/lmstudio`（设为 current）；前端面板按钮调它。

## 都不该做

- 改 src/provider/registry.ts 源码
- 重新 npm run build
- 直接 `unset OPENAI_API_KEY` 强制 chain fallback（fallback 只在错时触发，不是切换机制）
