# ASK-069 参考答案

| 命令 | 范围 | 持久化数据 |
|---|---|---|
| `/clear` | 当前 TUI 屏幕 / 显示 buffer | 不动 |
| `/forget --all` | 业务表（data.db）：dedup / cost / approvals / sessions metadata | 删（audit.db 写 tombstone） |

## /clear

- 作用层：显示
- 数据：sessions/<id>/messages.jsonl 仍在，下次进 session 看得到
- 用途：屏幕滚太长，重新整理视野

## /forget

- 作用层：数据
- audit 链不会断（写 tombstone 事件）
- 用途：真正"忘记"业务历史（如客户隐私要求）

## 关系

`/clear` ≠ `/forget`。
- `/clear` 不影响下次启动
- `/forget` 即使关 codeclaw，下次进也看不到了
