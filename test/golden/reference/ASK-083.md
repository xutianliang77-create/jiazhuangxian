# ASK-083 参考答案

## 推荐命令

```bash
sqlite3 -header -column .codeclaw/data.db \
  "SELECT session_id, sum(amount) AS total FROM cost GROUP BY session_id ORDER BY 2 DESC LIMIT 10;"
```

通过 bash 工具直接跑，stdout 即结果。

## 输出格式选项

| flag | 用途 |
|---|---|
| `-header` | 列名作为首行 |
| `-column` | 美观对齐（默认 list 格式） |
| `-csv` | CSV，方便 pipe |
| `-json` | JSON 数组 |
| `-html` | HTML 表 |
| `-line` | 每条 record 多行（适合宽行） |

## 不要

- 装 mysql / postgresql：codeclaw 用 SQLite
- 用 read 读 .db：二进制，read 拿不出 SQL 结果
- 在 codeclaw 内嵌 sql 执行：bash + sqlite3 是更简单的路径

## 进阶

如果 SQL 很长，可以先写 `.sql` 文件再 `sqlite3 ... < query.sql`：
```bash
echo "SELECT ..." > /tmp/q.sql
sqlite3 -header -column .codeclaw/data.db < /tmp/q.sql
```
