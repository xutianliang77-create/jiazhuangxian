# ASK-021 参考答案

## body_path 的语义

ObservationRepo 在 db 里存的 `body_path` 总是相对 `sessions/{sid}/`，例：

| 场景 | body_path |
|------|-----------|
| 正文 ≤ 100 KB（合并 jsonl） | `observations.jsonl` |
| 正文 > 100 KB（溢出） | `observations/step-1.jsonl` |

## 为什么相对路径

### 跨机迁移友好
- `~/.codeclaw/sessions/{sid}/` 整个目录可以 `rsync` 到另一台机，不用回头改 db 任何字段
- 绝对路径 `/Users/alice/.codeclaw/sessions/...` 一变 home 就全 broken

### 多账户 / 共享盘场景
- 同一份会话快照可以在 alice / bob 两个 home 下都跑，因为路径相对 `sessionsDir` 解析

### 备份 + 恢复
- backup 出来的目录直接还原到 `~/.codeclaw/sessions/`，db 字段无需改写

### 测试场景
- 单测里 sessionsDir 用 `mkdtempSync` 临时目录，相对路径让 db 数据可以脏读后清理而不留绝对路径垃圾

## 代价

- 读取时必须拼 `path.join(sessionsDir, body_path)`；ObservationRepo 已封装。
- 直接对 db 字段 `cat` 看不到完整磁盘路径，需要 join；调试稍麻烦。

## 反例

- "SQLite 不能存路径" → 错，TEXT 任意字节
- "一定要 symlink" → 不需要，相对路径已经够
