# ASK-099 参考答案

```sql
-- 方法 1：CTE + LEFT JOIN
WITH d0 AS (
  SELECT DISTINCT user_id
  FROM events
  WHERE event_ts::date = DATE '2026-04-01'
),
d7 AS (
  SELECT DISTINCT user_id
  FROM events
  WHERE event_ts::date = DATE '2026-04-08'
)
SELECT
  ROUND(
    100.0 * COUNT(DISTINCT d7.user_id) / NULLIF(COUNT(DISTINCT d0.user_id), 0),
    2
  ) AS d7_retention_pct
FROM d0
LEFT JOIN d7 USING (user_id);
```

## 方法 2：单 query（更快，少一次扫表）

```sql
SELECT
  ROUND(
    100.0 * COUNT(DISTINCT u.user_id) FILTER (
      WHERE EXISTS (
        SELECT 1 FROM events e2
        WHERE e2.user_id = u.user_id
        AND e2.event_ts::date = DATE '2026-04-08'
      )
    ) / NULLIF(COUNT(DISTINCT u.user_id), 0),
    2
  ) AS d7_retention_pct
FROM events u
WHERE u.event_ts::date = DATE '2026-04-01';
```

## 关键点

| 点 | 说明 |
|---|---|
| `event_ts::date` | 把 timestamp 截到日；如果 events 已有 date 列直接用 |
| `DISTINCT user_id` | 同一用户当天多次活动不重复计数 |
| `LEFT JOIN` | 保留 d0 全集（不在 d7 的也算分母） |
| `NULLIF(_, 0)` | d0 为 0 时除零 → NULL，避免 SQL 错 |
| `100.0 *` | 强制浮点除法，避免整数除法返 0 |
| `ROUND(_, 2)` | 保留 2 位小数 |
