# ASK-060 参考答案

## 步骤

```bash
# 1. 跑完整 vitest，看哪些测试 fail
npm test
# 或：npx vitest run

# 2. 看详细 reporter
npx vitest run --reporter=verbose

# 3. 跑单文件
npx vitest run test/path/to/file.test.ts

# 4. 按 test 名过滤
npx vitest run -t "match string"

# 5. 看 typecheck 是否也挂了
npm run typecheck
```

## 不要做

- `git push --no-verify`：绕开 gate 不修问题，本地 ok 但 CI / 队友拉下来仍坏
- 改 hook 配置降低门槛：除非用户明确同意

## pre-push gate 内容

`scripts/git-hooks/pre-push` 跑：
- `npm run typecheck`（tsc --noEmit）
- `npm test`（vitest run）

任一失败即拦。
