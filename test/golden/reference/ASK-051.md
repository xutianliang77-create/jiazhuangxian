# ASK-051 参考答案

## 组件名

`SafeTextInput`（自实现，替代 `ink-text-input`）

## 为什么不用 ink-text-input

| 问题 | 表现 |
|---|---|
| ink 把多次按键合并为单事件 | 用户连续按 backspace 只删一个字符 |
| `key.flag`（如 `key.backspace`）不可靠 | 需兜底匹配 raw 字节 `\x7f` / `\x08` |
| 与 ink 主循环互扰 | 输入字符乱跳 / 状态错位 |

修复路径见 commit `45c7f62` / `a8ac160`。`SafeTextInput` 自维护 buffer 与 cursor，监听 raw stdin。
