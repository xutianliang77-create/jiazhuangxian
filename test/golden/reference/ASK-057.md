# ASK-057 参考答案

## batch dir 命名

格式：`yyyymmdd-HHMMSS-rand4`（如 `20260425-114321-a8f2`）—— 时间戳 + 4 字符随机后缀

存放位置：`<workspace>/.codeclaw/backups/<batch-dir>/...`

## 缓存策略

```ts
let cachedBatchDir: string | null = null;
```

- 同一进程内，第一次写时生成名字并缓存；后续所有写复用 → **同一次运行**的所有 backup 都在同一个 batch dir 下
- 测试通过 `resetBackupBatch()` 清缓存

## workspace 外目标

例如 `backupFileIfExists('/etc/hosts', workspace)`，目标在 workspace 外：
- 加前缀 `_outside_/`，再用绝对路径相对 root 拼接
- 例如：`<workspace>/.codeclaw/backups/<batch>/_outside_/etc/hosts`
- 避免与 workspace 内同名相对路径冲突

## 阈值

- `MAX_BACKUP_FILE_BYTES = 10 * 1024 * 1024`（10 MB）：超过跳过备份（防大文件吃磁盘）
- 不存在 / 不是 file 也跳过
