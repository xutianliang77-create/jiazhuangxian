# ASK-017 参考答案

## 参考代码

```ts
import Database from "better-sqlite3";

const db = new Database(":memory:");
db.pragma("journal_mode = WAL");

const row = db.prepare("SELECT 1 AS one").get() as { one: number };
console.log(row);   // { one: 1 }

db.close();
```

## 跑法

```bash
tsx ./snippet.ts
# { one: 1 }
```

## 评分要点

- 必须用 `better-sqlite3`（**不是** `sqlite3`/`node-sqlite3`，那是另一个包，异步回调 API 不同）
- 必须 `import` 语法（ESM 友好），不接受 `require(...)`
- `db.pragma("journal_mode = WAL")` 写法
- 用 `prepare(...)` 然后 `.get()` 取一行
- 收尾 `db.close()` 加分（不挂连接）

## 常见扣分

- 用 `db.exec("SELECT 1")`：能跑但拿不到结果
- 用了 `node-sqlite3` 的 `Database.open` 异步 API：包错了
- 漏 `db.close()`：内存 db 无所谓，落到磁盘 db 时是问题
