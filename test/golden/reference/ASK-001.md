# ASK-001 参考答案

## 答题要点

CodeClaw 的 `/mode` 命令支持 **6 种权限模式**（对应 `src/lib/config.ts:PermissionMode`）：

1. `default`：常规交互；高风险动作必须确认
2. `plan`：只产出计划，不执行任何副作用
3. `auto`：自动允许 low/medium 风险；high 才 ask
4. `acceptEdits`：自动允许 write 类修改；bash 等仍 ask
5. `bypassPermissions`：全部自动允许（仍写审计）
6. `dontAsk`：按缓存决策，无则拒绝

## 评分要点

- 答案必须完整列出 6 种
- 不应杜撰 `admin` / `sudo` 等不存在的模式
- 可简短解释各模式用途（加分项）

## 常见扣分

- 少列任一模式：-1
- 多列 `admin` / `sudo`：直接 fail（命中 must_not_mention）
