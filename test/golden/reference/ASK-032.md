# ASK-032 参考答案

## 答题要点

Skill manifest 里 `commands: SkillSlashCommand[]` 字段定义 **该 skill 暴露的 slash 命令**。loader 启动时把每条 `name`（必须以 `/` 开头）注册到 slash registry；用户在对话里键入该命令，等价于 `/skills use <skill-name>` —— 用该 skill 启动一次会话。

### 最小例子

```yaml
# ~/.codeclaw/skills/lint-fix/manifest.yaml
name: lint-fix
version: 1.0.0
description: Auto-fix lint errors in TypeScript files
prompt: |
  你是一个 TypeScript lint 修复助手...
commands:
  - name: /lint
    description: 修复当前项目的 lint
```

用户输入 `/lint` ⇨ 等价于 `/skills use lint-fix`，启动 lint-fix skill 的对话会话。

### 校验规则

- name 必须 `/^\/[a-z][a-z0-9_-]{0,30}$/i`
- 不能与 builtin slash 命令（`/mode`、`/cost`、`/forget` 等）重名 —— loader 启动时报错并跳过

## 评分要点

- 答出 `commands` 字段、必须 `/` 开头、handler 等价于 `/skills use ...`
- 不应混淆为"执行 shell / bash 命令"

## 常见扣分

- 把 commands 当 shell command 执行：直接 fail
- 漏说 "/" 前缀强制：-1
