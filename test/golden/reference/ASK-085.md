# ASK-085 参考答案

## Release 前 5 件

### 1. 全量测试 + typecheck

```bash
npm run typecheck && npm test
```
任一失败 → 不发布。

### 2. Build 完整

```bash
npm run build
ls dist/storage/migrations/data dist/storage/migrations/audit
ls dist/public  # web 静态
```
关键：dist/ 含 SQL migrations + web 静态（build.mjs 显式 cp）。

### 3. Version 同步

```bash
grep '"version"' package.json
# "version": "0.5.0"

head -20 CHANGELOG.md
# ## 0.5.0 - 2026-04-25
#   - ...
```
package.json 与 CHANGELOG 最新条目要对得上。

### 4. Dry-run pack

```bash
npm pack --dry-run
# 输出：会包含哪些文件、总大小
```
确认：
- dist/ 完整
- migrations/ 在
- README.md 在
- 不带 .codeclaw / .env / sessions（应该 npmignore）

### 5. 实际打包并扫描

```bash
npm pack
tar -tf codeclaw-0.5.0.tgz | head -50
```
看清单，确保没把测试 / fixture 一起发。

## 通过后

```bash
npm publish --access public
# 或 私有：npm publish --otp=<2FA>
```

## 不要做

- `npm publish --force` 强发
- 跳过 test：发出去坏的版本要 deprecate / unpublish 麻烦
- 改 version 不更 CHANGELOG：用户搞不清新版本到底改了什么
