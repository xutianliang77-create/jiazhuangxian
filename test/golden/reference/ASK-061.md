# ASK-061 参考答案

| 类别 | 验证 | 修复 |
|---|---|---|
| 全局未装 | `npm list -g --depth=0 \| grep codeclaw` | `npm install -g .`（在仓库根） / `npm link` |
| 本地装但忘 link | `npm bin` 看 bin；`ls node_modules/.bin/codeclaw` | `npx codeclaw doctor` 直接跑 |
| PATH 漏 npm global bin | `echo $PATH \| tr : '\n'` 看是否含 `npm prefix -g`/bin | `export PATH="$(npm prefix -g)/bin:$PATH"` 加到 shell rc |

## 验证安装路径
```bash
which codeclaw    # 已装时返回路径
npm prefix -g     # global 安装根
ls $(npm prefix -g)/bin/ | grep codeclaw
```

## 不该做
- 重启系统：跟 PATH 无关
- `sudo apt install`：codeclaw 不在 apt 仓库
