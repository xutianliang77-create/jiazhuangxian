# ASK-055 参考答案

## 两类校验

| 维度 | 字段 | 限制 |
|---|---|---|
| 长度 | `prompt` | `MAX_PROMPT_LEN = 8000` 字节，超长 reject |
| 长度 | `description` | `MAX_DESCRIPTION_LEN = 500` 字节，超长 reject |
| 字符 | 两者皆 | `FORBIDDEN_CONTROL_CHARS` 正则禁 NUL / ANSI escape (\x1b) / 多数 control 字节 |

## 防御目的

1. **Prompt injection（T6 供应链）**：恶意 manifest 在 prompt 里塞 `\x1b[2J` 清屏 + 假装 system prompt 输出 / 用 NUL 截断显示 / Unicode bidi override 反转语义
2. **资源耗尽**：超长 prompt 撑爆上下文 / token 预算
3. **输出毒化**：control char 让用户终端显示被篡改

不是 SQL / XSS（manifest 不进 db / 不渲染 HTML）。
